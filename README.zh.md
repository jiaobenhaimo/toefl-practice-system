# 超能录 在线模考系统

支持用户认证、服务端评分和管理面板的在线模拟考试平台。目前配置为 TOEFL，通过 `config.yaml` 可扩展至其他考试类型。

## 快速开始

```bash
pip install -r requirements.txt
python app.py
```

默认管理员：**admin / admin**，请立即修改密码。

## 用户角色

**管理员** — 管理账号、查看成绩、分配测试。**教师** — 查看成绩、分配测试。**学生** — 在 `/assignments` 完成分配的测试，在 `/catalog` 自由练习。**访客** — 免登录匿名练习。

## 导航

桌面端显示固定侧边栏，移动端使用汉堡菜单。根据角色显示不同链接。

## 测试分配

教师在 `/teacher/results` 分配测试，学生在 `/assignments` 查看并完成。学生也可在 `/catalog` 自由选择练习。

## 配置

`config.yaml` 配置站点名称、考试类型和部分定义，为未来扩展到其他考试类型做准备。

## 功能

- SQLite 数据库、服务端评分、侧边栏导航
- 测试分配、练习模式、中英文界面
- 深色模式、PDF 导出、每题计时
- 进度点 + 书签、ARIA 无障碍
